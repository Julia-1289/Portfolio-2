# Моё портфолио / сайт-визитка

Это мой личный сайт-визитка, созданный в рамках учебного проекта. Цель - показать навык верстки, работы с препроцессором Sass и JavaScript, а также понимание accessibility и адаптивной верстки.

## Используемые технологии и подходы

- **Чистый JavaScript** — без библиотек и фреймворков, для демонстрации понимания основ.
- **Препроцессор Sass** — использую переменные, миксины, утилиты и функции для организации стилей и сокращения кода.
- **Адаптивная верстка** — сайт корректно отображается на всех устройствах (ПК, планшет, смартфон).
- **Темная и светлая темы** — реализованы с помощью CSS-переменных и переключения стилей.
- **Accessibility (доступность)** — стараюсь учитывать все важные требования: семантическая разметка, контроль фокуса, читаемость и использование атрибутов ARIA при необходимости.
- **Методы организации кода** — использование миксинов и утилит для упрощения и поддержки кода.
- **Git и GitHub** — контроль версий и публикация сайта.
